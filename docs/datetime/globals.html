timeToString

  • timeToString(milliseconds: number, formatString?: string): string
    • Defined in packages/datetime/src/operators/timeToString.ts:67

    Return string representation of time span according to specified format.

    Format can be one of the predefined specifiers: c, g, G or custom in the way hh:mm:ss.MMM.

    Interval Format Result
    1.14:30:15 c 1.14:30:15
    1.14:30:15 g 1:14:30:15
    1.14:30:15 G 1.14:30:15.000
    1.14:30:15 hh:mm:ss 14:30:15
    1.14:30:15 hh:mm:ss.MMM 14:30:15.000

    Parameters

    • milliseconds: number

      time span in milliseconds

    • Optional formatString: string

    Returns string
